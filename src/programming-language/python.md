# Python

## Anaconda

创建环境：

```sh
conda create --name py3 python=3.7

# conda create --name py2 python=2.7
```

激活环境：

```sh
conda activate py3

# conda activate py2
```

## PyTorch

```sh
conda install pytorch torchvision cudatoolkit=10.0 -c pytorch
```

> 参考于 <https://pytorch.org/get-started/locally/>。
